# docs.html 样式优化说明

## 更新概览

已将 docs.html 的样式调整为更朴素、更统一的 GitHub 风格，同时保持所有现有功能。

## 主要改进

### 1. Tab 样式统一化

**改进前：**
- 两级 Tab 使用不同的背景色和圆角
- 有较大的内边距和阴影效果
- 激活状态使用白色背景和阴影

**改进后：**
- 统一使用透明背景
- 移除圆角和阴影效果
- 使用简单的底部边框（橙色 #fd8c73）表示激活状态
- 两级 Tab 样式完全一致，看起来更像一体

```css
/* 统一的 Tab 样式 */
- padding: 8px 16px (更紧凑)
- font-size: 14px (标准 GitHub 字体大小)
- border-bottom: 2px (激活时显示橙色边框)
- 无背景色、无阴影
```

### 2. 整体间距优化

**改进：**
- 减小了各区块的 margin 和 padding
- 表格单元格从 8px 减小到 6px
- 代码块从 1rem 减小到 12px
- Section 标题间距从 2rem 减小到 1.5rem

### 3. 字体大小调整

**改进：**
- Tab 按钮：14px（标准）
- API 详情标题：1.75rem（之前 2rem）
- Section 标题：1.25rem（之前 1.5rem）
- 表格字体：12px（表头）、13px（内容）
- 页脚：12px

### 4. 颜色方案简化

**Badge 徽章：**
```css
GET:    #ddf4ff 背景 + #0969da 文字
POST:   #dafbe1 背景 + #1a7f37 文字  
PUT:    #fff8c5 背景 + #9a6700 文字
DELETE: #ffebe9 背景 + #cf222e 文字
```

**API 类型徽章：**
```css
WebAPI: #ddf4ff 背景 + #0969da 文字
C API:  #ffebe9 背景 + #cf222e 文字
```

**JSON 语法高亮（适配浅色背景）：**
```css
键名:   #0969da（蓝色）
字符串: #0a3069（深蓝）
数字:   #953800（橙色）
布尔值: #8250df（紫色）
null:   #cf222e（红色）
```

### 5. 移除过度动画

**改进：**
- 移除 Tab 切换的 fadeIn 动画
- 移除 API 详情的 fadeIn 动画
- 保留必要的 hover 过渡效果（0.2s）

### 6. 页脚优化

**改进：**
- 背景从深色（#1e293b）改为浅灰色（var(--sidebar-bg)）
- 文字颜色从浅色改为次要颜色（var(--secondary-color)）
- 内边距从 2rem 减小到 1.5rem
- 字体大小从 0.9rem 减小到 12px

## 样式对比

### Tab 激活状态

**之前：**
- 白色背景 + 阴影
- 蓝色底部边框
- 粗体字重（700）
- 圆角顶部

**现在：**
- 透明背景
- 橙色底部边框（#fd8c73）
- 中等字重（600）
- 无圆角

### 代码块

**之前：**
- 1rem 内边距
- 较深的文字颜色（深色模式配色）

**现在：**
- 12px 内边距
- 适配浅色背景的语法高亮
- 更紧凑的间距

## 功能保持

✅ 两级 Tab 切换功能完整保留
✅ WebAPI 和 C API 类型切换正常
✅ 请求、回执、流程图、注意事项切换正常
✅ 所有表格、代码块、徽章功能正常
✅ 响应式布局未受影响
